Sobre mí

Hola, soy Rafael de Barranquilla. Soy un ingeniero de sistemas apasionado por la tecnología y el desarrollo de software. Me gradué de la Universidad de la Costa y me encanta aprender nuevas tecnologías y aplicarlas en proyectos innovadores.

Proyectos

Proyecto 1: Sistema de Gestión de Inventarios

Un sistema para gestionar inventarios de una tienda, incluyendo funcionalidades de seguimiento de productos, control de stock y generación de reportes.

Proyecto 2: Aplicación de Tareas

Una aplicación web para gestionar tareas diarias, con funcionalidades de creación, edición y eliminación de tareas, así como recordatorios y notificaciones.

Proyecto 3: Sitio Web Personal

Un sitio web personal para mostrar mi portafolio, incluyendo información sobre mí, mis proyectos y formas de contacto.
